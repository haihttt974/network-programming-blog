Python

Ngôn ngữ lập trình đa dụng, phổ biến trong backend, data và AI

1. Giới thiệu

Python là ngôn ngữ lập trình bậc cao, cú pháp đơn giản, tập trung vào khả năng đọc hiểu.

2. Đặc điểm kỹ thuật

  • Paradigm: Multi-paradigm (OOP, Procedural, Functional)
  • Typing: Dynamic, Strong
  • Execution: Interpreted
  • Garbage Collection: Có

3. Hệ sinh thái

  • Web: Django, Flask, FastAPI
  • Data / AI: NumPy, Pandas, PyTorch, TensorFlow
  • Automation: Scripts, DevOps tools

4. Use case phổ biến

  • Backend API
  • Data analysis
  • Machine Learning
  • Automation

5. Ưu điểm

  • Dễ học
  • Cộng đồng lớn
  • Thư viện phong phú

6. Hạn chế

  • Hiệu năng thấp hơn C/C++
  • Không phù hợp real-time system

7. Lộ trình học đề xuất

  1. Cú pháp cơ bản
  2. OOP
  3. Virtual environment & packaging
  4. Web / Data specialization
